# exStickGE_microblaze

## Requirements
- Vivado 2020.1
- Vitis 2020.1

## Build

```
vivado -mode batch -source ./create_project.tcl
```

You can get `prj/prj.runs/impl_1/top_wrapper.bit` and `prj/top_wrapper.xsa`.
